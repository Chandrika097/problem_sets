Problem Sets

#

Solutions in a variety of languages to various programming challenges from around the web such as UVa, Project Euler, etc.

Literate documentation is generated using pycco.

#

Notes

1. While this project is MIT licensed, it's the intention of the author to sharpen his own saw and as such encourages you to attempt the problems personally, only using these as examples for reference.

2. UVa problems generally read from stdin, but since I have no intention of submitting these solutions they vary in what forms of input they accept. Some read from file, some from a term prompt, it just depends on the alignment of the planets and what is currently residing in my emacs kill ring.

#

Obtaining
